# VUNA-Calc

A simple yet powerful calculator that displays computational results in English language. Perfect for learning number pronunciation or verifying calculations with word representations.

## Features

- **Basic Arithmetic Operations**: Addition, subtraction, multiplication, and division
- **Number-to-Words Conversion**: Automatically converts calculation results to English words
- **Decimal Support**: Handles decimal numbers with proper word representation
- **Bracket Support**: Use parentheses for complex calculations
- **Backspace Functionality**: Easily correct input mistakes
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Clean UI**: Built with Bootstrap for a modern, intuitive interface

## Demo

The calculator displays results in two formats:

- **Numeric**: Standard numerical output
- **Words**: English language representation (e.g., "123" becomes "one hundred twenty three")

## Technologies Used

- HTML5
- CSS3 (Bootstrap 5)
- Vanilla JavaScript

## Getting Started

### Prerequisites

No special prerequisites needed! Just a modern web browser.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/VUNA-Calc.git
   ```

2. Navigate to the project directory:

   ```bash
   cd VUNA-Calc
   ```

3. Open `index.html` in your web browser:

   ```bash
   # On Linux
   xdg-open index.html

   # Or simply double-click the index.html file
   ```

## Usage

1. Click number buttons (0-9) to input numbers
2. Click operator buttons (+, -, \*, /) to perform operations
3. Click "=" to calculate the result
4. View the result in both numeric and word format
5. Use "AC" to clear all inputs
6. Use "←" (backspace) to delete the last character
